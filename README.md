# nvim-config

A simple, but intuitive, Neovim Configuration setup built from the ground up using the Lazy Neovim package manager.

**Configured By:** <a href="https://github.com/mohrammadr09">Mohammad</a>

<hr />

# Installation & Using my Configuration

To install Neovim, run the following command: 
```bash
# macOS
brew install neovim # using homebrew to install neovim
```

For Windows & Linux machines, look up the documentation for appropriate installation methods and directions.

Afterwards, run this command which will initialize a `nvim` config directory & git clone this repository.

```bash
# note that these instructions are tailored for macOS

cd ~/.config
git clone https://github.com/mohammadr09/nvim-config.git # clone the repository using HTTPS; if you use SSH or the GitHub CLI, feel free to replace this line with the appropraite command
mv nvim-config nvim
```
